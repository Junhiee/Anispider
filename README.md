# Anispider

## 技术栈

由 Python 技术栈完成 Anispider 项目

httpx - 请求库

playwright - 自动化

rq - redis 队列

motor (async) - mongodb 数据库

fastapi - 对外提供api接口 [full-stack-fastapi-template](https://github.com/fastapi/full-stack-fastapi-template)
